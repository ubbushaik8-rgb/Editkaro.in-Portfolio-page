# 🎬 Editkaro.in Portfolio Website

A modern, responsive portfolio website developed for **Editkaro.in**, showcasing the agency's creative work through an immersive and interactive user experience. Built using **HTML5**, **CSS3**, and **Vanilla JavaScript**, the website combines elegant design, smooth animations, and dynamic portfolio interactions to create a professional digital presence.

---

# 📖 Overview

The Editkaro.in Portfolio Website is a responsive single-page application designed to showcase video editing services, creative projects, and digital content solutions. The project focuses on delivering a visually appealing interface with intuitive navigation, interactive portfolio sections, and a seamless browsing experience across desktop, tablet, and mobile devices.

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/ubbushaik8-rgb/Editkaro.in-Portfolio-page.git
```

## Navigate to the Project

```bash
cd Editkaro.in-Portfolio-page
```

## Run Locally

Open `index.html` directly in your browser, or start a local server.

```bash
python -m http.server 8000
```

Then visit:

```
http://localhost:8000
```
