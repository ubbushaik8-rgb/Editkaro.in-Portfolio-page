# 🎬 Editkaro.in Portfolio Website

A modern, responsive portfolio website developed for **Editkaro.in**, a creative agency specializing in video editing and digital content creation. Built using **HTML5**, **CSS3**, and **Vanilla JavaScript**, the website delivers an engaging user experience through interactive portfolio showcases, smooth animations, responsive layouts, and modern UI design.

---

## 🌐 Live Demo

**🔗 https://editkaro.netlify.app**

---

# 📖 Overview

The **Editkaro.in Portfolio Website** is a fully responsive single-page web application designed to showcase the agency's creative services, featured projects, and digital expertise. The project focuses on delivering a visually appealing interface with intuitive navigation, immersive animations, and a seamless browsing experience across desktop, tablet, and mobile devices.

The application follows modern frontend development practices with clean code architecture, reusable components, and performance-focused implementation.

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/ubbushaik8-rgb/Editkaro.in-Portfolio-page.git
```

## Navigate to the Project

```bash
cd Editkaro.in-Portfolio-page
```

## Run the Project

Open **index.html** directly in your browser.

For a local development server:

```bash
python -m http.server 8000
```

Visit:

```text
http://localhost:8000
```
